
# Number Guessing Game

A Simple number guessing game written in java.

# how to run
1.Clone the repository : git clone http:
2.Open the project in vs.code or your preferred IDE.
3.Run the 'numberguessinggame.java' file.


# game description 
The game generated a random number between 1 to 100 , and the player has 5 attempts to guess it correctly . after each guess, the game provides feedback in the formof " too high" or "too low".

# code explanation 
The game uses a simple loop to generate the random number and get user input . 
It uses conditional statements to provide feedback and determine the game's outcome.

# Features 
*Random number geration
*User input validation
*feedback mechanism
*win/lose conditions
